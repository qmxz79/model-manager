# Model Manager Skill

## 名称
`model-manager` - 模型管理器

## 描述
提供图形界面来管理 OpenClaw 的模型和供应商，无需在终端运行命令。

## 功能
1. **查看当前模型** - 弹出对话框显示当前配置
2. **选择模型** - 在对话框中点击选择模型
3. **添加供应商和模型** - 在对话框中添加新的供应商和模型
4. **删除供应商和模型** - 在对话框中删除已设置的供应商和模型

## 使用方法

### 1. 打开模型管理器
```
查看当前模型
打开模型管理器
模型管理
```

### 2. 在对话框中操作
- **选择模型**：点击模型列表中的模型
- **添加模型**：填写表单并点击"添加模型"
- **删除模型**：点击模型旁边的"删除"按钮
- **删除供应商**：点击供应商旁边的"删除供应商"按钮
- **应用更改**：点击"应用更改"按钮保存设置

## 界面功能

### 对话框界面
1. **当前默认模型** - 显示当前使用的模型
2. **可用模型列表** - 所有可用模型的列表
3. **供应商管理** - 显示和管理供应商
4. **添加新模型** - 表单用于添加新模型

### 操作按钮
- **应用更改** - 保存模型设置
- **刷新** - 刷新页面
- **关闭** - 关闭对话框

## 技术实现

### 文件结构
```
model-manager/
├── SKILL.md              # 技能说明
├── ui-manager.js         # 图形界面管理器
├── ui-server.js          # Web 服务器
├── model-manager.js      # 核心功能（保留）
└── references/           # 参考文档
```

### 启动方式
1. **Web 服务器**：运行 `ui-server.js` 启动本地服务器
2. **访问地址**：`http://localhost:8081`
3. **对话框界面**：自动在浏览器中打开

## 注意事项
- 修改配置后需要重启 OpenClaw 才能生效
- 确保 API 密钥环境变量已设置
- 建议备份原始配置文件以防万一
