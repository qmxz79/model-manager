# 模型管理器 UI 使用指南

## 启动界面

### 方法1：双击启动
双击 `start-ui.bat` 文件启动 Web 服务器

### 方法2：命令行启动
```bash
cd C:\Users\Administrator\.openclaw\workspace\model-manager
node ui-server.js
```

### 方法3：通过 OpenClaw 启动
```
打开模型管理器
查看当前模型
```

## 界面功能

### 1. 查看当前模型
- 顶部显示当前默认模型
- 点击模型可切换默认模型

### 2. 模型列表
- 显示所有可用模型
- 点击模型选择
- 点击"删除"按钮删除模型

### 3. 供应商管理
- 显示所有供应商
- 显示供应商的模型数量
- 点击"删除供应商"按钮删除供应商

### 4. 添加新模型
- 填写供应商名称
- 填写模型ID和名称
- 选择API类型
- 点击"添加模型"按钮

## 操作流程

### 切换模型
1. 在模型列表中点击想要的模型
2. 模型会高亮显示
3. 点击"应用更改"按钮

### 添加模型
1. 在"添加新模型"区域填写表单
2. 点击"添加模型"按钮
3. 新模型会出现在模型列表中

### 删除模型
1. 在模型列表中找到要删除的模型
2. 点击模型旁边的"删除"按钮
3. 确认删除

### 删除供应商
1. 在供应商管理区域找到要删除的供应商
2. 点击"删除供应商"按钮
3. 确认删除（将删除该供应商下的所有模型）

## 注意事项

1. **重启生效**：修改配置后需要重启 OpenClaw 才能生效
2. **备份配置**：建议备份原始配置文件
3. **端口占用**：如果8081端口被占用，可修改 `ui-server.js` 中的 PORT

## 快捷访问

- **本地访问**：`http://localhost:8081`
- **查看当前模型**：`http://localhost:8081?action=view`
- **管理模型**：`http://localhost:8081`
